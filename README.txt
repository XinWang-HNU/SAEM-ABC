This folder contains sample code replicating experiments presented in 
Umberto Picchini and Adeline Samson (2017): Coupling stochastic EM and Approximate Bayesian Computation for parameter inference in state-space models, arXiv:1512.04831.

Contact the first author for enquiries.

Content:
- the "nonlinear-gaussian" folder pertains section 4.1 in the cited paper.
- the "theophylline" folder pertains section 4.2 in the cited paper.
